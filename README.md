# BassirAI Automation

Automation workflows for the BassirAI AI-powered patient communication and appointment booking platform.

---

## Overview

This repository contains the automation layer of the BassirAI platform.

BassirAI is an AI-powered patient communication and appointment booking platform designed for aesthetic clinics. The automation system connects AI agents, messaging channels, scheduling, databases, and external APIs to deliver a seamless patient experience.

This repository serves as the central location for automation workflows, documentation, prompts, templates, and integration scripts.
